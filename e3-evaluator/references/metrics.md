# E3 Reference Metrics (v1.0)

## E1: Effectiveness (Software Usefulness)
*Focus: Solving the right problem for the right people.*

| Metric | Target | Description |
| :--- | :--- | :--- |
| **Task Completion Rate** | $\geq 0.90$ | Successful tasks / Total attempted tasks. |
| **Intent Alignment Score** | $\geq 0.85$ | Degree to which outputs match stakeholder intent. |
| **Hallucination Rate** | $\leq 0.05$ | Factual errors / Total outputs. |
| **Ethical Compliance** | $1.00$ | Binary check for safety/fairness violations. |

## E2: Efficiency (Logic and Performance)
*Focus: Solving it correctly and fast.*

| Metric | Target | Description |
| :--- | :--- | :--- |
| **Inference Latency** | $\leq 200ms$ | Time from request to response. |
| **Tokens per Query** | Minimize | Efficiency of prompt/completion token usage. |
| **Carbon Intensity** | Minimize | Energy source optimization (Carbon-aware). |
| **Retry Rate** | $\leq 0.10$ | Regenerations / Total requests. |

## E3: Elegance (Interface and Experience)
*Focus: Solving it beautifully and maintainably.*

| Metric | Target | Description |
| :--- | :--- | :--- |
| **System Usability Scale**| $\geq 68$ | Standardized usability questionnaire score. |
| **Explanation Coverage** | $\geq 0.80$ | Outputs with verifiable reasoning. |
| **Actionability** | $\geq 4.0/5.0$ | Usefulness of provided explanations. |
| **Format Compliance** | $\geq 0.95$ | Adherence to standardized schemas (e.g., MCP). |
