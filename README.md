> **"Stop manually tweaking hyperparameters. Start programming the researcher."**

The **E3 Framework** is a standardized protocol for software engineering in the age of Agentic AI. It provides a formal system for balancing **Effectiveness** (Software Usefulness), **Efficiency** (Logic & Performance), and **Elegance** (Interface & Experience).

This repository is **Agent-Native**: it is designed to be read, implemented, and iterated upon by AI agents (e.g., Gemini CLI, Claude Code) with minimal human intervention.

## 🚀 Quick Start (For Agents)

If you are an agent, follow these steps to perform an E3 evaluation:

1.  **Clone the Repo**: `git clone https://github.com/niaj/e3-framework`
2.  **Run the Scorecard**: `python3 e3-evaluator/scripts/e3_scorecard.py <your_project_dir>`
3.  **Perform E3 Checks**: Consult the [framework/](framework/) directory for specific , E_2, E_3$ protocols.

## 📚 Repository Structure

- [**framework/**](framework/): Refined E3 definitions and "E3 Checks" for the 2026 SDLC.
- [**e3-evaluator/**](e3-evaluator/): The `.skill` package and Python-based quality scorecard.
- [**paper/**](paper/): Foundational research paper LaTeX sources (Academic Review, Formal Definitions).
